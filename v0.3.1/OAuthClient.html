Class: OAuthClient

OAuthClient(clientId, clientSecretopt)

An OAuth 2.0 client, encapsulating the client ID and optional client secret. (Both are generated by MediaWiki when the client is created.)

Constructor

new OAuthClient(clientId, clientSecretopt)

Parameters:
Name Type Attributes Default Description
clientId string

The public client ID, sometimes known as the consumer token in MediaWiki. Usually 32 hexadecimal characters.
clientSecret string | null <optional>
 null

The client secret, sometimes known as the secret token in MediaWiki. Usually 40 hexadecimal characters. Can be omitted or set to null for non-confidential clients; however, non-confidential clients can also still use a client secret: see T323867 for discussion.
